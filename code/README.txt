This is code of analysis correlation.

Language Version:Python 3.X
System Version:macOS Mojave 10.14.5

Need to plugin Jacoco and PItest pachage to root.pom of data files. Based on configutation.

There are four source code files:
1. Correlation between (branch, statement)coverage and complexity coverage.
2. Correlation between (branch, statement)coverage and mutation score.
3. Correlation between (branch, statement)coverage and post-release defect density.
4. Correlation between code change and post-release defect density.

To run the source codes, just click run in menu on the top for each code file or find the main function and click main function to run.

After run the codes, each file will generate according png files.
